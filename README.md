**Este sitio web fue desarrollado a pedido de una clienta que buscaba una invitación online personalizada para su 50° cumpleaños.** El objetivo principal fue centralizar toda la información del evento y gestionar la confirmación de invitados de manera ágil y moderna.

## 📌 Descripción del proyecto

El proyecto consiste en una **Single Page Application (SPA)** diseñada bajo el enfoque **Mobile First**, entendiendo que el 100% de los invitados accedería al sitio a través de sus dispositivos móviles (vía WhatsApp).

A diferencia de proyectos académicos, este se trabajó con una **clienta real**, lo que permitió ajustar la estética y funcionalidad según requerimientos específicos: desde la armonía visual hasta la lógica del panel de administración.

## 📱 Experiencia de Usuario & Compartibilidad

Dado que el principal canal de difusión fue a través de redes sociales, se puso especial énfasis en la **coherencia visual externa**:

- **Open Graph & Meta Tags:** Implementación exhaustiva de etiquetas en el `<head>` para asegurar que, al compartir el link por WhatsApp o redes, se genere una previsualización atractiva (con título, descripción e imagen miniatura).
- **Diseño Estético Coherente:** Se utilizaron recursos visuales seleccionados para mantener una línea de diseño profesional y festiva en todo el sitio.
- **Enfoque Mobile:** Interfaz optimizada para pantallas táctiles, asegurando que botones y formularios sean fáciles de usar desde cualquier smartphone.

## ⚙️ Tecnologías utilizadas

Para este desarrollo se priorizó la velocidad de carga y la funcionalidad, utilizando un stack eficiente:

- **HTML5** → Estructura semántica y optimización de metadatos sociales.
- **CSS3** → Diseño responsivo personalizado y estilos visuales.
- **JavaScript (Vanilla)** → Interactividad, manejo de la cuenta regresiva y lógica del registro.
- **Firebase** → Utilizado como base de datos para la gestión de invitados en tiempo real.
- **GitHub Pages** → Hosting del sitio, ofreciendo una solución estable y de bajo mantenimiento.

## 🛠️ Apartado Admin

El sitio cuenta con una sección dedicada (`admin.html`) donde se detalla la **lista de asistencia**. Esta funcionalidad permite a la organizadora visualizar en tiempo real quiénes han confirmado, permitiendo un control total sobre la logística del evento.

## 🚀 Estado del proyecto

✅ **Funcionalidad Completa:** El sitio cumple con todos los requisitos solicitados.
✅ **Social-Ready:** Optimizado para una previsualización visual impactante al ser compartido.
✅ **Despliegue Exitoso:** El sitio está live y operativo en GitHub Pages.
